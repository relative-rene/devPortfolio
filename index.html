{Full-Stack Developer && Bob Ross Enthusiast}
When do you become a Full-Stack Developer. I suppose I dabbled with idea when I was able to design a web application on Wireframes and implement it with JavaScript on a MongoDB database. must have basked in my success for all of a minute, before I was in my terminal hitting 'rails new' and began the journey of accesskey="" It must have been, a I am tentative to call myself a Full-Stack Web Developer. Yes, alone and with a team, I have built functional web application from wire-frames to deployment. that I call myself a Full-Stack Web Developer. Looking for a creative, self starter who has a passion for web design? Call off the search, you have found the man for the job. After 1000+ hours of structured and unstructured learning, I have built web based applications from wireframes to deployment via Heroku. Cutting my teeth in an Agile environment, I am comfortable seeking answers, failing quickly and contributing tested code with a Git workflow. After recently completing General Assembly’s Web Development immersive course, I have a proven ability to learn a new technology and contribute functional transparent code. General Assembly’s WDI is a 12-week intensive course focused on working with databases such as Postgres, MySQL ,MongoDB, and languages/frameworks such as JavaScript, Ruby on Rails, Angular, Node.js and Express..
